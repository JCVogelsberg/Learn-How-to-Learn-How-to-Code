Skills

FINDING ANSWERS: #1 skill a dev can have. No-one can remember it all; there's too much and it changes regulary. You need to be at least a blue belt in Google Fu.


TENACITY: You're going to feel stupid for a good long while. It's a steep learning curve, just stick with and you'll realize you're learning a lot.


REGULAR PRACTICE: The only way to learn to code is to code. You have to practice regularly. Reading isn't going to cut it, watching tutorial videos won't cut it. You've got to practice regularly. Another parallel to martial arts.
